6. Zoek op of experimenteer wat "+=" doet. Werkt deze ook met Strings? Weet je hoe je deze anders kan schrijven?

+= is een Addition Assignment Operator. De waarde aan de rechterkant wordt opgeteld bij de waarde aan de linkerkant. Deze operator werkt ook met Strings.

Een voorbeeld met Number: let x = 25; x += 10; console.log(x) De uitkomst hiervan is 35.

Een voorbeeld met Strings: let text = "Hoi"; text += " allemaal"; console.log(text)

7. Doe hetzelfde voor "-=", "*=", "/=", "%=" en "**=".

-= is een Subtraction Assignment Operator. Deze trekt de waarde af van de variable.

Een voorbeeld:let x = 20; x -= 4; console.log(x). De uitkomst hiervan is 16.


*= is een Multiplication Assignment Operator. Deze vermenigvuldigd de variable.

Een voorbeeld: let x = 5; x *= 5; console.log(x). De uitkomst hiervan is 25.


/= is een Division Assignment Operator. Deze deelt de variable.

Een voorbeeld: let x = 100; x /= 2; console.log(x). De uitkomst hiervan is 50


%= is een Remainder Assignment Operator. Deze kent een restant toe aan de variable.

Een voorbeeld: let x = 10; x %= 2; console.log(x). De uitkomst hiervan is 0. Een tweede voorbeeld: let x = 9; x %= 2; console.log(x). De uitkomst hiervan is 1.


**= is een Exponentiation Assignment Operator. Deze verheft de variable tot de macht van de waarde.

Een voorbeeld:let x = 10; x **= 3; console.log(x). De uitkomst hiervan is 1000.
