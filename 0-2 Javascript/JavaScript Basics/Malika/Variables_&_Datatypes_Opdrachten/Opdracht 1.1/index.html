Opdracht 1.1

We onderzoeken het verschil tussen var, let en const. Gegeven zijn een aantal voorbeelden - schrijf eerst je verwachtingen op voordat je ze zelf nabootst. Kijk vervolgens wat er bij jou in de console gelogd wordt - is dat wat je verwachtte? Zo nee, deel het dan aan het einde van de dag met je groep.

a)

Ik verwacht dat er in de console “Let Tekst Twee” gelogd wordt omdat let kan veranderen en ik verwacht dat de laatste wordt gelogd.

Verwachting was fout. Ik krijg de volgende foutmelding: Uncaught SyntaxError: Identifier 'letOefening' has already been declared.

b)

Ik verwacht dat er in de console zowel “Let Tekst” als “Let Tekst Twee” wordt gelogd omdat bij beide teksten nu console.log staat.

Verwachting was goed.

c)

Ik verwacht dat er in de console alleen “Const Tekst” wordt gelogd omdat bij de tweede geen const voor staat.

Verwachting was fout. Ik krijg de volgende foutmelding: Uncaught TypeError: Assignment to constant variable.

d)

Ik verwacht dat er in de console zowel “Var Tekst Twee!” als “Var Tekst Drie!” worden gelogd. Ik verwacht niet dat “Var Tekst” wordt gelogd omdat er geen var voor staat.

Verwachting was fout. In de console zijn "Var Tekst" en "Var Tekst Twee!" gelogd. "Var Tekst Drie is niet gelogd."
