Opdracht 1

  • Block box

    • De box komt elke keer op een nieuwe regel terecht.
    • De width en height eigenschappen worden gerespecteerd.
    • De waarde die padding, margin en border hebben, veroorzaken dat andere items weg worden geduwd van de box.
    • Wanneer de width niet gespecificeerd wordt dan neemt het element de ruimte in die zijn container heeft op inline niveau.

  • Inline box

    • De box blijft op dezelfde regel.
    • De width en height eigenschappen worden niet toegepast.
    • Verticale margin, padding en border worden toegepast alleen worden andere inline boxes niet verplaatst daardoor.
    • Horizontale margin, padding en border worden toegepast en zorgen ervoor dat andere inline boxes worden verplaatst.

Opdracht 4

  • Content: Dat wat in een element zit.
  • Padding: De ruimte tussen de content en de border.
  • Margin: De lege ruimte om een box die andere items wegduwd.
  • Border: De omlijning van de Padding en de content.
