Opdracht 1

De drie manieren om CSS in te laden zijn:

  • Inline: Door de stijlregels te gebruiken in een html element.
  • Internal: Door middel van een style element te definieren in het html document en daar de stijlregels in te definieren.
  • External: Door in de Header te linken naar een externe file waarin alle stijlregels worden gezet.

Opdracht 2

De syntax van html bestaat uit een selector, gevolgd door een declaration block. De selector selecteert de items waarop de declarationblock van toepassing is. De declaration block bestaat uit meerdere declarations die zelf weer bestaan uit een property, een eigenschap die veranderd wordt en een value, dat hoe de property wordt veranderd.

Opdracht 3

    1. Voorbeeld type selectors: De h1 elementen van deze pagina
    2. Voordelen: Je kan dezelfde elementen allemaal tegelijk selecteren
    1. Voorbeeld class selectors: De voorbeeld list items binnen deze opdracht
    2. Voordelen: Je kan een style toevoegen die voor verschillende elementen hetzelfde moeten zijn maar niet voor alle typen. Je kan ze ook delen over verschillende soorten typen elementen.
    1. Voorbeeld id selectors: De voordelen list item van dit type selector.
    2. Voordelen: Je kan specifieke stijlen toewijzen.
    1. Voorbeeld descendant selectors: de p elementen die volgen na een h1 element op deze pagina.
    2. Voordelen: Je kan gericht in een specfieke structuur stylen toewijzen.
