Opdracht 1

De drie manieren om CSS in te laden zijn:

  1. Inline: Door de stijlregels te gebruiken in een html element.
  2. Internal: Door middel van een style element te definieren in het html document en daar de stijlregels in te definieren.
  3. External: Door in de Header te linken naar een externe file waarin alle stijlregels worden gezet.

Opdracht 2

De syntax van html bestaat uit een selector, gevolgd door een declaration block. De selector selecteert de items waarop de declarationblock van toepassing is. De declaration block bestaat uit meerdere declarations die zelf weer bestaan uit een property, een eigenschap die veranderd wordt en een value, dat hoe de property wordt veranderd.
