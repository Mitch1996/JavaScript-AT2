Opdracht 1

Je kunt op 3 verschillende manieren CSS in je HTML inladen.

Op welke 3 manieren kun je dat doen?

Leg ook voor elke wijze uit wat het precies inhoud.

De CSS kun je op 3 manieren koppelen aan je website:

Inline style: CSS-code in de HTML-tag.

Interne stylesheet: CSS-code als blok in de head van de pagina.

Externe stylesheet: CSS als los bestand.
