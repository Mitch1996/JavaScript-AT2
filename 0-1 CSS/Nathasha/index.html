Opdracht 2

CSS pas je toe met een bepaalde syntax

Hoe ziet zo'n syntax eruit?

Kun je ook uitleggen wat elk element uit de syntax betekent?

CSS heeft een bepaalde syntax.

Dit zijn de regels die je moet volgen bij het schrijven van css.

Elke verwijzing van CSS bestaat uit minstens 3 onderdelen:

de selector
de property
een waarde

id (identity)

Je kan aan elke HTML-tag een id toewijzen.

Dit is een persoonlijke naam voor deze tag om deze tag specifiek aan te spreken.

Je geeft hier dan je H1 HTML-tag een id en verwijst in je CSS bestand naar deze id

door voor de naam van de selector een ‘#‘ te plaatsen.

Normaal gesproken komt een id maar een keer voor, deze is dus uniek!
