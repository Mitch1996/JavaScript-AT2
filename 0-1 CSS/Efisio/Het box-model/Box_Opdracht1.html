Opgave 1

  • Block boxes : Altijd onder vorig block element, breedte hangt af van parent container en hoogte hangt af van de inhoud
  • Inline boxes : vertical spacing heeft geen invloed, ze dienen om elementen te highlighten in een block. de breedte hangt af van de inhoud

    Opgave 4

  • Content: de tekst, image, of andere media in het element
  • Padding: de ruimte tussen de box inhoud en de border
  • Margin : de ruimte tussen de box en de omringende boxes
  • Border : de lijn tussen de box padding en de margin
    •
