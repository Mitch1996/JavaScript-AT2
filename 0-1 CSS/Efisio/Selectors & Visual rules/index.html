  • External CSS

Opdracht 2

CSS pas je toe met een bepaalde syntax. Hoe ziet zo'n syntax eruit? Kun je ook uitleggen wat elk element uit de syntax betekent?

  • Selector : Selector geeft aan welk HTML element je wilt bewerken, kan elke tag zijn zoals h1, div etc.
  • Declaration block : Declaratie block heeft een of meer declaraties die worden gescheiden door een puntkomma. Declaratie heeft eigenschapnaam en bijbehorende waarde.
  • Property : is een attribuut van een HTML element, kan color, border zijn.
  • Value : Waarde die toegewezen is aan de property, waarde kan bijvoorbeeld "yellow" hebben als toegewezen property.

Opdracht 3

Er zijn verschillende selectors, geef voorbeelden. Geef in je antwoord ook aan wat de voordelen zijn van elke selector.

Voorbeeld CSS Type selector :

Voordeel Type selector is dat je niet steeds de attributen en bijbehorende values hoeft op te geven voor een bepaald Type, declareer je eenmaal

Voorbeeld CSS Class selector :

Voordeel Class selector is dat je niet steeds de attributen en bijbehorende values hoeft op te geven voor een bepaalde Class, declareer je eenmaal

Nog een voordeel is dat je zelf een Classe kunt maken met toepasselijke naam

Voorbeeld CSS ID selector :

Voordeel ID selector is dat je niet steeds de attributen en bijbehorende values hoeft op te geven voor een bepaalde ID, declareer je eenmaal

Nog een voordeel is dat je zelf een custom ID kunt maken met toepasselijke naam en attributen

Voorbeeld CSS Descendant selector :

Voordeel Descendant selector is dat bepaalde elementen matched van een bepaald bovenliggend element en hier attributen en values mee kunt geven.
