Display & Positioning

Opdracht 1

  • Position static: Dit is de default waarde, elementen worden niet beinvloed door top,bottom,left en right properties
  • Position relative: Dit betekend dat het element een relatieve positie heeft ten opzichte van zijn normale positie
  • Position absolute: Dit element is relatief gepositioneerd ten opzichte van de dichtstbijzijnde ancestor
  • Position fixed: Dit element blijft op dezelfde positie, zelfs als de pagina scrollt
  • Position sticky: Dit element blijft op dezelfde relatieve positie ten opzichte van de pagina die eventueel voorbij scrollt
