    • Opdracht 2: CSS pas je toe met een bepaalde syntax. Hoe ziet zo'n syntax eruit? Kun je ook uitleggen wat elk element uit de syntax betekent?

    • Selectors: Dit onderdeel duidt aan waar de opgemaakte tekst of stijlblok zich bevindt. Deze plaats kun je op 3 verschillende mogelijke manieren aanspreken.
    • Properties: Het is ook perfect mogelijk om meerdere properties toe te kennen aan 1 selector. Zoals een tekst in het vet en cursief te plaatsen. We scheiden hiervoor alle propertys van elkaar met ; .
    • Values: Waarde vertelt aan CSS hoe je wilt dat het element veranderd

    Opdracht 3

    • Type selectors- Bijvoorbeeld om alle paragrafen te centreren en in het rood dan ziet het als volgt uit: p { text-align: center; color: red; }
    • Class selectors- Begint met een . en selecteert alles in het document waar de class op wordt toegepast. Bijvoorbeeld: .highlight { background-color: yellow; }
    • ID selectors - Wordt op dezelfde manier gebruikt als de class selector. Hierbij begin je met een# ipv een . bijvoorbeeld: #one { background-color: yellow; } h1#heading { color: purple; }
    • Descendant selectors- Om elementen binnen een element te stijlen. Bijvoorbeeld: p b { color: Blue; }
