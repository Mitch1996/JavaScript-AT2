      • Type selectors- Bijvoorbeeld om alle paragrafen te centreren en in het rood dan ziet het als volgt uit: p { text-align: center; color: red; }
      • Class selectors- Begint met een . en selecteert alles in het document waar de class op wordt toegepast. Bijvoorbeeld: .highlight { background-color: yellow; }
      • ID selectors - Wordt op dezelfde manier gebruikt als de class selector. Hierbij begin je met een# ipv een . bijvoorbeeld: #one { background-color: yellow; } h1#heading { color: purple; }
      • Descendant selectors- Om elementen binnen een element te stijlen. Bijvoorbeeld: p b { color: Blue; }
