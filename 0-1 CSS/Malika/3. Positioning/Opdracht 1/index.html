Opdracht 1

De belangrijkste waardes van “position” zijn static, relative, absolute, fixed & sticky. Leg voor elke uit wat ze doen.

  • Static: Elementen zijn gepositioneerd volgens de normale flow van de html pagina.
  • Relative: Elementen zijn net als bij static volgens de normale flow van de html pagina gepositioneerd. Echter kun je hier veranderingen brengen in de top, right, bottom en left properties.
  • Absolute:Elementen worden verwijderd uit de normale flow van de html pagina. Ook hierbij kun je gebruik maken van de top, right, bottom en left properties.
  • Fixed:Elementen worden verwijderd uit de normale flow van de html pagina. Echter in tegenstelling tot absolute beweegt het element mee als je scrollt waarbij het element op dezelfde positie blijft. Ook hierbij kun je gebruik maken van de top, right, bottom en left properties.
  • Sticky: Is een combinatie van relative en fixed.
