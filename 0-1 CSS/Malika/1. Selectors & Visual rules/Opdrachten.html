Je kunt op 3 verschillende manieren CSS in je HTML inladen. Op welke 3 manieren kun je dat doen? Leg ook voor elke wijze uit wat het precies inhoud.

Je kunt CSS op de volgende drie manieren inladen in je HTML:

  1. External: Hierbij maak je gebruik van een externe stylesheet. In de HTML-pagina zet je een referentie in de element, in de head.
  2. Internal: Hierbij defineer je de stijl in de stijl element in de head.
  3. Inline: Hierbij voeg je een style attribute toe aan een specifiek element.

Opdracht 2

CSS pas je toe met een bepaalde syntax. Hoe ziet zo'n syntax eruit? Kun je ook uitleggen wat elk element uit de syntax betekent?

Elke CSS regel bestaat uit een selector en een declaration block. De selector verwijst naar het HTML element dat je wilt stijlen. De declaration block bevat meerdere declarations die onderscheiden worden door puntkomma's. Elke declaration bevat een CSS property name en value, onderscheiden door een dubbele punt.

Opdracht 8

Styling die je toepast wordt toegepast in een bepaalde volgorde. Het kan voorkomen dat je styling toepast en vervolgens niet het gewenste resultaat ziet. Cascade en Inheritance zijn belangrijke concepten om te begrijpen hoe styling werkt. Lees meer over Cascade en Inheritance. Zoek ook andere referenties om het concept te begrijpen. Leg in je eigen woorden uit wat met cascade en inheritance wordt bedoeld. Maak voorbeelden om je antwoord uit te leggen.

CSS is een afkorting voor Cascading Style Sheets. Een letterlijke vertaling voor cascading is trapsgewijze (proces dat in stappen verloopt). Het kan voorkomen dat je iets invoert in CSS en vervolgens een andere uitkomst hebt dan je had verwacht. Dit kan meerdere oorzaken hebben. Zo kan er al een algemene style zijn toegepast op dat element. Of door inheritance heeft een element al een bepaalde value overgenomen. Als er tegenstrijdige invoeren zijn voor een element geldt er een trapsgewijze proces waarbij de ene regel de ander overruled. Zo kan de volgorde een rol spelen waarbij de invoer lager in het document 'wint'(gewenste uitkomst); hoe dichter bij het element hoe hoger de priority. Ook kan specifity een rol spelen waarbij de invoer die het meest gespecifeerd is 'wint'. Verder zou je bijvoorbeeld !important in je invoer kunnen zetten. Dat zal alles overrulen maar wordt zeer afgeraden.
