Opdracht 1

Er zijn 14 verschillende pseudo-elementen. Beschrijf ze alle 14 kort en maak voor elk een voorbeeld.

  1. After: Hierbij wordt iets toegevoegd achter het gekozen element.

  2. Before: Hierbij wordt iets toegevoegd voor het gekozen element

  3. First-Line: Hiermee kun je de eerste regel stijlen. ------------------------------------------------------------------------------------------------ ------------------------------------------------------------------------------------------------ --------------------------------------------------------------------- Dit geldt dus niet voor overige regels.

  4. First-Letter: Hiermee kun je de eerste letter stijlen.

  5. Backdrop: Hieronder zie je een voorbeeld met een dialog.

    6. I'm a dialog window
  6. Cue:Hierbij worden WebVTT cues binnen een geselecteerd element gematched en gestijld.

  7. File-selector-button: Hiermee kun je het element input met type file stijlen.


  8. Grammar-error: Hiermee kun je het gedeelte dat grammar errors heeft stijlen.

  9. Highlight:Hiermee kun je een custom highlight stijlen.

  10. Marker: In plaats van een cijfer voor elke listitem is de stijl nu veranderd naar een *.

  11. Part():Hiermee stijl je elementen in een shadow tree met een matching part attribute.

  12. Placeholder: Hiermee stijl je de placeholder text in een input of textarea.



  13. Selection: Hiermee stijl je geselecteerde gedeeltes van een tekst.

    14. In dit voorbeeld kun je zien dat woorden die je selecteert een roze achtergrond krijgen.


  14. Spelling-error: Hiermee kun je het gedeelte dat spellingsfouten heeft stijlen.

  15. Target-text: Hiermee stijl je tekst waar naar toe gescrold is.
