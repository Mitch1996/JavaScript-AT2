Opdracht 2

CSS pas je toe met een bepaalde syntax. Hoe ziet zo'n syntax eruit? Kun je ook uitleggen wat elk element uit de syntax betekent?

De syntax bestaat uit een selector en een declaratie blok. De selector wijst naar het HTML element dat je wilt stijlen. In het declaratie blok vind de daadwerkelijke stijling plaats.

Opdracht 3

Er zijn verschillende selectors, bijvoorbeeld:

  • type selectors
  • class selectors
  • ID selectors
  • descendand selectors
TYPE SELECTOR Stijl selector staat in het HTMl element. In onderstaand voorbeeld wordt voor element ‘h1’ de kleur rood gebruikt en lettergrootte 23 pt. Deze stijl geldt voor ieder ‘h1’-element. h1 { color: #FF0000; size: 23pt; }
ID SELECTORS en CLASS SELECTORS

Een ID selector en een Class geven beide de opmaak van een bepaald element in de CSS weer. Bij ID gebruik je een hekje # voor de naam, terwijl dit bij class een punt is. Het grote verschil is, dat je een Class meerdere keren kunt gebruiken in de pagina en een ID uniek is en dus maar 1 keer gebruikt mag worden in een pagina.

Een id gebruik je als het gaat om een unieke naam die gekoppeld is aan hoofdelementen van een website. Bijvoorbeeld div id=” header”, en zo ook voor: de navigatie, de inhoud, de zijkant, de footer en de zoekfunctie. Deze onderdelen komen maar een keer op een webpagina voor en krijgen daarom een unieke naam

Een class gebruik je om individuele onderdelen te koppelen aan CSS. Je wilt bijvoorbeeld een tekst rood maken en dan geef je deze tekst een class mee. Deze class kan vaker worden gebruikt in een webpagina. Je kunt namelijk besluiten om meerdere delen van de tekst rood te maken. Voorbeeld: .class1{ color: black;} .class2{ color: red;}

De descendant selector ("afstammeling selector") stelt je in staat om je te beperken tot elementen die afstammeling (descendant) zijn van een ander element. De syntaxis is erg eenvoudig, je geeft de naam van de parent (ouder), één of meer spaties, en dan de naam van het element dat je eigenlijke doel is. Hier volgt een voorbeeld:

p b { color: Blue; } <p>Hello, <b>world</b> - what a <b>beautiful</b> day!</p>

Cascade bepaalt welke CSS-regel wint indien meerdere regels van toepassing zijn op een element. Cascade voorkomt herhaling in CSS code. Inheritance bepaalt welke properties worden geërfd door de child elementen van het element waar die properties op worden toegepast (en welke niet). Specificity bepaalt welke selector wint indien meerdere selectors op hetzelfde element van toepassing zijn.
