Opdracht 1

Block box

  • De box gaat iedere keer naar een nieuwe regel
  • With en Height eigenschappen worden gerespecteerd
  • Padding, margin and border zorgen ervoor dat andere elementen van de box worden weggeduwd
  • Indien width niet wordt gespecificeerd dan neemt het element de ruimte in van zijn box op inline niveau

Inline box

  • De box blijft op dezelfde regel
  • with en height eigenschappen worden niet toegepast
  • Verticale padding, margin en border worden toegepast, andere inline boxes worden daardoor niet verplaatst
  • Horizontale padding, margins en borders worden toegepast, andere inline boxes worden daardoor wel verplaatst

Opdracht 4

  • Content: datgene wat in het element zit, de inhoud.
  • Padding: ruimte tussen content en border.
  • Marging: de marge, lege ruimte om de box die andere items weggeduwd.
  • Border: omlijning van content en padding.
