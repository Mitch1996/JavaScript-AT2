Opdracht 1

  • Static: De normale opmaak van de pagina word behouden, er zijn geen aanpassingen.
  • Relative: Het element verplaats je door middel van de properties "TOP", "LEFT', "RIGHT" en "BOTTOM".
  • Absolute: Element word geplaats ten opzichte van het dichtbijzijnde ander element
  • Fixed: Element is gefixeerd op dezelfde plek, ook al scroll je de pagina.
