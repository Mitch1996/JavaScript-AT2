Dit zijn de opdrachten voor CSS

Opdracht 1

Je kunt op 3 verschillende manieren CSS in je HTML inladen. Op welke 3 manieren kun je dat doen? Leg ook voor elke wijze uit wat het precies inhoud.

  1. In de body van het html bestand kun je een style sectie maken, met daarin de opdrachten uit CSS
  2. In de head van het html bestand kun je een link invoeren naar een apart CSS bestand. Door een link met daarin rel en href
  3. In de heaad van het html bestand kun je ook opmaak invoeren die dan voor het gehele bestand geldt

Opdracht 2

CSS pas je toe met een bepaalde syntax. Hoe ziet zo'n syntax eruit? Kun je ook uitleggen wat elk element uit de syntax betekent?

Een Syntax bepaalt hoe we code schrijven in CSS. Het bestaat uit drie elementen/regels: een selector, een property en een value.

Opdracht 3

Er zijn verschillende selectors, bijvoorbeeld; type selectors, class selectors, ID selectors en descendant selectors

Maak van de bovengenoemde selectors voorbeelden. Geef in je antwoord ook aan wat de voordelen zijn van elke selector.

  • type selectors: Het selecteert alle elementen die behoren tot hetzelfde type. bijv p of h1
  • class selectors: wordt gebruikt om alle elementen te selecteren, die horen bij dezelfde class attribute. bijv. class = red
  • ID selectors: matcht een element met een bepaalde value. bijv. id=value
  • descendant selectors: iedere selector met lege spatie erin. bijv h1{ }
