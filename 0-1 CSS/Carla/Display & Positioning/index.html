CSS opdrachten Display & Positioning

Opdracht 1

  • static: Is de default setting. Is niet op een speciale manier gepositioneerd
  • relative: Gedraagt zich hetzelfde als static, tenzij je nog een aantal extra properties opneemt, bijv: top,right,left bottom.
  • absolute: lastigste waarde voor position. Gedraagt zich zet als fixed, behalve als deze relatief tot de dichtbijzijnde gepostioneerde voorouder (parent) element in tegenstelling tot relatief in de viewport.
  • fixed: Is realtief gepositioneerd ten opzichte van de viewport, hoidt in dat element altijd op dezelfde plek staat ongeacht of pagina wordt gescrolld. Er worden ook top, right, left ,bottom properties gebruikt
  • stickyzijn altijd relatief ten opzicte van het document, niet de parent. Wordt niet beinvloed door scrollen
